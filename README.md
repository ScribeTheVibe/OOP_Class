**Studentų rezultatų skaičiuoklė**

Programa turi 3 veikimo metodus:
1. Duomenų suvedimas ranka
2. Duomenų suvedimas ranka, išskyrus pažymius kurie yra atsitiktinai sugeneruoti
3. Visų duomenų atsitiknis sugeneriavimas

Programa paskaičiuoja rezultatus pagal formulę -
Rezultatas = (pažymių suma / pažymių kiekio) * 0,4 + egzamino rezultatas * 0,6.

Programa taip pat paskaičiuoja mediana.
